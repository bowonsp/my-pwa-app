Cara mendapatkan URL: File → Share → Publish to web → CSV format

📋 Instruksi Setup Google Sheets

1. Buat Google Sheet baru dengan kolom header berikut:

  • ID - Nomor unik
  • Nama - Nama data
  • Email - Email terkait
  • Kategori - Jenis data
  • Jumlah - Nilai numerik
  • Tanggal - Tanggal input
  • Status - Status data
  • Keterangan - Catatan tambahan

2. Publish Sheet: File → Share → Publish to web → pilih "CSV" → Publish

3. Copy URL CSV dan paste ke form di atas
